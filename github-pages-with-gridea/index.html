如何使用 GitHub Pages 托管网站

本文是关于使用 GitHub Pages + Gridea (Web) 部署网站的教程。全程无需编程或写代码，所以无论你是从事什么行业都可以跟随此教程部署一个自己的网站。

1. 第一步：注册 GitHub 账号
注册一个 Github 账号，用于登录 GitHub，以使用 GitHub 仓库功能

2. 第二步：创建仓库

仓库是用来存储网站静态文件的，如 HTML 文件

登录 GitHub，然后点击右上角的加号或点击此链接创建一个仓库名为[你的用户名].github.io]的仓库

例如你的用户名为Xiaoming，则仓库名为Xiaoming.github.io，注意仓库名中的.github.io后缀

注意：仓库类型需为 Public

3. 第三步：创建 Token

Token 是用来推送网站文件到仓库的

点击此链接进行个人访问令牌(Personal access token)的创建。权限只需要勾选 repo 中的 public_repo 即可。

创建成功后，复制令牌（Token）到本地，稍后将在 Gridea 配置中使用。

注意：令牌（Token）只在创建成功时显示一次，所以尽快复制到本地

4. 第四步：配置 Gridea
打开你的 Gridea 后台，将仓库 URL和令牌
填入，然后点击检查连接并保存。当看到成功的提示时，说明你已经配置完成。
